# Flaskr

The basic blog app built in the Flask tutorial.


## Install

Create and activate a virtual environment:

~~~sh
python3 -m venv venv
. venv/bin/activate
~~~

Install the Flaskr application:

~~~sh
pip install -e .
~~~


## Run

Run the `run_app.sh` script, then open http://127.0.0.1:5000/ in a browser.
